# Dashboard de Análisis de Retención y Satisfacción Estudiantil

Autores: Juan Varela, Carlos Forero

## Descripción
Este repositorio contiene una aplicación en Streamlit que analiza la tasa de retención y la satisfacción estudiantil,
y compara los términos Spring y Fall.

## Archivos
- `app.py` : aplicación Streamlit.
- `requirements.txt` : dependencias.
- `Informe_Actividad1.pdf` : resumen con hallazgos.

## Cómo ejecutar localmente
1. Instala dependencias:
   ```
   pip install -r requirements.txt
   ```
2. Ejecuta la app:
   ```
   streamlit run app.py
   ```

## Notas
- El archivo de datos `university_student_data (1).csv` debe colocarse en la misma carpeta o ajustar la ruta en `app.py`.
